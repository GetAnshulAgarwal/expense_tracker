## Project Setup Instructions

1. **Clone the repository**
   ```
   git clone <repository-url>
   ```

2. **Navigate to the project directory**
   ```
   cd assignment_cs
   ```

3. **Install dependencies**
   ```
   flutter pub get
   ```

4. **Run the application**
   ```
   flutter run
   ```
